# Java Rest Api Client Example for Oro Applications

## Prepare

Replace host, api key and the user name in `src/main/java/com/oroinc/api/ServiceGenerator.java`


### Build

```
./gradlew build
```

### Create Jar

```
./gradlew fatJar
```

### Run Jar

```
java -jar build/libs/api-fat-1.0.jar
```
